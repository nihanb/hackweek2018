## Description

hackweek 2018 project

## Setup

```sh
# environment & build tools
brew install node yarn # node.js & yarn
```

**Dependencies**

```
yarn install
```

**Run**

```bash
yarn start
```

**Test**

```bash
# unit
yarn test
```
